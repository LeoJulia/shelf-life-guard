# О проекте

Это веб-приложение для умной организации вашего косметического арсенала. Мы помогаем следить за сроками годности, контролировать расход средств и экономить бюджет, избегая дублирования покупок.

## Ключевые возможности:

1. 📦 **Умные карточки товаров**: Вводите данные о кремах, сыворотках, тонерах и другой косметике для лица и тела.
2. ⏳ **Трекер жизненного цикла**: Отслеживайте дату покупки, открытия баночки, текущий срок годности и момент завершения использования.
3. 📊 **Аналитика и статистика**: Визуализируйте свои расходы, узнайте, какие бренды вы покупаете чаще всего, и сколько средств осталось на полке.
4. ⚙️ **Гибкая настройка**: Настраивайте отображаемые поля под себя и используйте мощную фильтрацию по брендам, типам продуктов или статусу (открыто/закрыто).

Идеальное решение для тех, кто ценит порядок в своей рутине ухода за кожей.

# Установка

Установите зависимости проекта:

```bash
yarn install
```

## Запуск

Запустите сервер разработки на `http://localhost:3000`:

```bash
yarn dev
```

Соберите проект для продакшена:

```bash
yarn build
```
